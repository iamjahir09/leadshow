BACKEND (server/server.js) - Express on port 5000:

GET  /api/conversations                        → fetch from Chatwoot
GET  /api/messages/:conversationId             → fetch messages
POST /api/sendMessage                          → body: { conversationId, content }
POST /api/toggleStatus                         → body: { conversationId, status }

Use dotenv to read .env file. Use axios to call Chatwoot API.
Chatwoot API base: {CHATWOOT_URL}/api/v1/accounts/{ACCOUNT_ID}/
Header: api_access_token: {CHATWOOT_ACCESS_TOKEN}

server/package.json must have "type": "module" and use ES module imports.
server dependencies (npm): express, cors, dotenv, axios
